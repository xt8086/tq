Features

3.8x KV Compression

Compresses V cache to 4.25 bits — 8x more context in the same RAM.

Detects GPU, RAM, quant type. Calculates optimal cache settings.

OpenAI-Compatible

/v1/chat/completions. Works with curl, Pi, LM Studio.

Idle Auto-Stop

Stops after 5 min idle. No wasted resources.

Quick Start
  1. Install: curl -fsSL https://wondermotor.com/install.sh | bash (macOS) or pip install tq-serve; tq install (Windows)
  2. List models: tq list
  3. Search & download: tq search "qwen2.5 coder 7b" — pick a number to download
  4. Serve: tq serve 1 — TurboQuant auto-configured.
